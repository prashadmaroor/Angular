About the Website


There are three main Section in the website

  • Home Section: This section gives an detailed overview of the Website
  • User List : This section illustrates the GET HTTP request which retrieves all the users and DELETE HTTP request which deletes a particular user
  • Register : This section illustrates the POST HTTP request which is used to add user details to the Json server.

Hidden Sections

There are two hidden sections in the website

  • Hidden Section #1 : This section is used to illustrate GET/Posts/1 the read only operation and will be launched on click of data in Id Column of User List Section
  • Hidden Section #2 : This section is used to illustrate PUT/posts/1 the update request and will be launched on click of Edit button in Update column of User List section .Also Id field is read only. Id field is to ensure that there no update is made for invalid id and to ensure info of user selected is only updated

Toaster Service is integrated to show neccessary notifications
